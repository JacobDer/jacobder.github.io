View full project here

I chose facial recognition as my first independent foray into deep learning as well as the PyTorch and Keras frameworks. Ultimately I decided on using PyTorch since I preferred its lower-level approach. The data I used for this project was also collected and arranged entirely by myself. This project still has potential for more work and learning opportunities, as I believe architecture improvements and a massive dataset would help its performance.
