Covid-19 Dashboard

July 15, 2020
Jacob Der

This is a reactive dashboard I made using python to create a choropleth of the US and display graphs of covid-19 cases/deaths over time. Over on my github you can also go view the code I used to produce this dashboard.

The data used comes from the New York Times, and can be found here.

Something to take note of might be the fact that in quite a few states the relationship between Covid-19-related deaths and time appears astonishingly linear, meaning that the rate (first derivative) at which the coronavirus is killing isn't slowing down there. Still in others, the concavity (second derivative) of the graph is positive, meaning that the rate at which the coronavirus is killing is increasing.
