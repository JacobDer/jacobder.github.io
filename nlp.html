Fake News Detection

View full project here

UPDATE: I am aware of the issues with this project—the metrics obtained clearly point to some set of words being unreasonably predictive. However, I am leaving it up for the time being just to serve as demonstration of the general idea for the project.

In this NLP/classification project, I trained a naive Bayes classifier to detect whether or not a news article is fake news, based on the article's headline. This was achieved by transforming a dataset of news articles into a bag-of-words representation (with the help of textbag) and training a multinomial naive Bayes classifier on the data. Data and description can be found here.

Bag-of-words toolkit

View full project here

This toolkit, tentatively named "textbag", is a small python library I made make preprocessing text data slightly easier. Specifically, there are functions for cleaning various types of text data (string, numpy, pandas), creating a set of all of the words in the text data, and for creating a bag-of-words representation of the text data. I've also published this library to PyPi, so it can be easily installed with pip.
